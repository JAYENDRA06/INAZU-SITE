# INAZU Website

Promotional landing page for INAZU, a performance-focused ride tracking and analytics platform for bikes and cars.

## Stack

- Next.js 16
- TypeScript
- App Router
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Notes

- The current CTA uses `hello@inazu.app` as a placeholder contact destination.
- The SVG visuals in `public/` are custom placeholders for the marketing site and can be replaced with final product imagery later.
